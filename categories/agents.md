# AI for Science Agents

This category tracks agent-style systems that use language models or multi-agent patterns for scientific research, tool use, planning, or reporting.

OpenSciFlow's position is complementary: agents can be the user-facing entry point, but safe scientific execution needs reviewed plugin manifests, workflow templates, validation tests, citations, and reproducibility records.
